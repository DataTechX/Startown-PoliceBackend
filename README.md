# Fastify Minimal Starter

Look at the [Fastify documentation](https://fastify.io/docs/latest/) to learn more.

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```
